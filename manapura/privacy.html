プライバシーポリシー

最終更新日: 2026年6月1日

まなぷら（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーでは、本アプリが収集する情報とその利用目的について説明します。

1. 収集する情報

1.1 アカウント情報

Apple Sign Inを通じて以下の情報を取得します。

  • Apple ID に紐づくユーザー識別子
  • メールアドレス（Apple がリレーする場合があります）

1.2 ユーザーが入力する情報

  • お子さまのニックネーム・生年月日
  • 進路の希望（公立/私立等の選択）
  • 貯蓄目標・記録

これらの情報は教育費シミュレーションおよび貯蓄管理の目的でのみ使用されます。

1.3 広告に関する情報

本アプリはGoogle AdMobを使用して広告を表示します。AdMobは広告配信のために広告識別子（IDFA）を使用する場合があります。トラッキングの許可はユーザーの選択に基づきます。

1.4 自動的に収集される情報

本アプリは、クラッシュログやアプリの使用状況に関する匿名化された情報を収集する場合があります。

2. 情報の利用目的

  • 教育費シミュレーション・貯蓄管理機能の提供
  • デバイス間のデータ同期
  • 広告の表示
  • アプリの改善

3. 情報の保管

ユーザーデータはSupabase（Amazon Web Services 東京リージョン）上に暗号化して保管されます。データベースへのアクセスはRow Level Security（RLS）により、本人のデータのみに制限されています。

4. 第三者への提供

以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません。

  • ユーザーの同意がある場合
  • 法令に基づく開示要求があった場合
  • サービス提供に必要な業務委託先（Supabase、Google AdMob）への提供

5. データの削除

ユーザーはアプリ内の設定画面からアカウントを削除できます。アカウント削除時には、サーバー上の全データが削除されます。

6. お子さまのプライバシー

本アプリは保護者が使用することを想定しており、13歳未満のお子さまが直接使用することを想定していません。お子さまに関する情報（ニックネーム・生年月日等）は保護者が入力し、教育費の計算目的でのみ使用されます。

7. ポリシーの変更

本ポリシーは必要に応じて変更されることがあります。重要な変更がある場合は、アプリ内で通知します。
